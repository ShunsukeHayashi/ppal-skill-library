# Article Writer

**Version**: 1.0.0
**Purpose**: Generate high-quality articles

---

## Workflow

### Step 1: Research

1. Identify target keyword
2. Search for related content
3. Find unique angle

### Step 2: Outline

```markdown
# [Title with Keyword]

## Introduction (Hook + Problem + Promise)

## [H2: Main Point 1]
### [H3: Sub-point]

## [H2: Main Point 2]

## [H2: Main Point 3]

## Conclusion (Summary + CTA)
```

### Step 3: Write

- 1500-2500 words
- Short paragraphs (2-3 sentences)
- Include examples and data

### Step 4: Optimize

| Element | Best Practice |
|---------|---------------|
| Title | 60 chars, keyword at start |
| Meta | 155 chars, include keyword |
| H1 | One per page, match title |
| Images | Alt text with keyword |

---

## Output Format

```markdown
---
title: "[Title]"
description: "[Meta description]"
keywords: ["keyword1", "keyword2"]
---

# [H1 Title]

[Content...]
```
